# 基础

## 统计文件中字数

你可以使用 wc 命令来获取一个文件中的总的行数，字数和字符数。以下是简单的示例来查看有关上面创建的文件的信息:

```bash
$ wc filename2 19 103 filename
```

这里是所有四个列的细节:
1. 第一列: 代表文件中的行数。2. 第二列: 代表文件中的字数。3. 第三列: 代表文件中的字符数。这是文件的实际大小。4. 第四列: 代表文件名。
